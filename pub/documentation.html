Guide.js

Getting Started

  • Download Guide.Js


    Download the library.

  • JavaScript - Required Files

    The files guide.js and guidejs.css are required to use this library.
    Include this script in the <head> of your html page 
                                    
            <link rel="stylesheet" type="text/css" href="./styles/guidejs.css" />
            <script defer type="text/javascript" src='./javascript/guide.js'></script>

  • JavaScript - Example

    
            const guide = new GuideJs()

            guide.setOptions({
              elementDescriptions: ['Button 1 - Shows First Step', 
                                    'Button 2 - Shows Second Step', 
                                    'Here is some Example Text to show Highlighting'],
              addOverlay: true,
              overlayOpacity: 1,
              addOutline: true,
              theme: "red",
              guideWidth: '500px',
              guideHeight: '300px',
              guideDistance: 30,
            })

            guide.start()

  • HTML - Example

                                   
            <div class="example" >
                <button class="guidejs-step-1">Button 1 </button >
                <button class="guidejs-step-2">Button 2 </button >
                <span class="guidejs-step-3" > Example Text </span >
            </div >

API - GuideJs

GuideJs.setOptions(settings)

settings Property Default Type Description
elementDescriptions [] Array Sets the description/information for the element the guide box is providing context for.
addOverlay true Boolean If true, adds a black tinted overlay, focusing in on the element and the guide box only.
If false, no overlay is added.
overlayOpacity 0.9 Number If addOverlay is true, this sets the opacity of the overlay.
0 to 1 range.
Low value means more transparent, high value means more opaque.
addOutline true Boolean If true, adds an outline to the
If false, no outline is added.
theme "blue" String If "blue", sets theme of the guide box (the outline and progress bar) to blue.
If "red", sets theme of the guide box (the outline and progress bar) to red.
If "green", sets theme of the guide box (the outline and progress bar) to green.
If "yellow", sets theme of the guide box (the outline and progress bar) to yellow.
guideWidth "250px" String The width of the guide box.
Note: Since the input value is a string, units other than px can be used (such as vw, %, etc.)
guideHeight "120px" String The height of the guide box.
Note: Since the input value is a string, units other than px can be used (such as vh, %, etc.)
guideDistance 20 Number Distance between the guide box and the element it is providing context for (in px)
Note: Recommended value is default value.

GuideJs.start()

Function to start the guide (after setting the options with GuideJs.setOptions(settings).

GuideJs.stop()

Function to manully stop the guide.
